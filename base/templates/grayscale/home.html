About FlyBee

FlyBee is a python microframework based on bottle and fully compatible with bottle microframework https://bottlepy.org. This project is under GPLv3 License.

FlyBee using gevent and greenlets as webserver see the project for more information http://www.gevent.org. FlyBee Required Python 3.x to run.
